####toggleStyles.js
Toggle styles on or off in browser.

#####Usage
Console:
`clearStyle()` toggles styles off.
`resetStyle()` toggles styles on.     
Keyboard:
Spacebar keydown toggles styles on or off.

####clearStyle.min.js
Toggle styles off in browser.

#####Usage
Console.
